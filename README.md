# QA Manual Testing Projects

This repository contains my practice projects related to manual testing.

I focused on testing simple applications like e-commerce and banking systems to understand how real testing looks like.

---

## What I tested

- login functionality  
- shopping cart  
- placing orders  
- basic banking operations (transfers)

---

## What I practiced

- writing test cases and test scenarios  
- checking different situations (valid and invalid data)  
- looking for bugs and edge cases  
- creating simple bug reports  

---

## Bug reports

You can find examples of bugs such as:
- missing validation  
- incorrect system behavior  
- problems with buttons or forms  

---

## Goal

My goal was to learn how to think like a tester and get some hands-on experience before my first job in QA.
