# Manual Testing Project – Banking Application

Projekt stworzony w ramach nauki QA. Symulacja testów aplikacji bankowej.  
Testy wykonane ręcznie w środowisku symulowanym.

## Zakres testów:
- logowanie użytkownika
- przelewy
- saldo konta
- historia transakcji
- wylogowanie

## Typy testów:
- testy funkcjonalne
- testy negatywne
- testy regresji

## Edge Cases:
- bardzo długie dane wejściowe
- szybkie wielokrotne kliknięcia
- brak internetu / timeout
- równoczesne operacje

## Uwagi dodatkowe:
- w niektórych przypadkach komunikaty systemu są niejasne
- czas ładowania strony ok. 3-4 sekundy w Chrome
